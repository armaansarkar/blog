[Stab](https://github.com/armaansarkar/stab) is a vibecoded Chrome extension that automatically kills tabs.

## Features

- Closes **inactive** tabs
- Closes **duplicate** tabs
- Closes tabs **hogging memory**

## Install

1. Clone the repo
2. Go to `chrome://extensions`
3. Enable Developer mode
4. Click "Load unpacked" and select the Stab folder
5. Configure via the extension icon
