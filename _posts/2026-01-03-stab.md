[Stab](https://github.com/armaansarkar/stab) is a Chrome extension that automatically manages tabs to keep your browser fast.

## Features

- **Idle detection** - automatically closes tabs that haven't been active for a configurable time (default: 30 minutes)
- **Duplicate detection** - closes duplicate tabs while preserving the most recently used version
- **Memory detection** - for Chrome 116+, detects and closes memory-intensive tabs

## Install

1. Clone the repo
2. Go to `chrome://extensions`
3. Enable Developer mode
4. Click "Load unpacked" and select the Stab folder
5. Configure via the extension icon
